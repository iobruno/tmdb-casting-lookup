# tmdb-casting-lookup
TMDB Casting Lookup

## Tech Stack
- Python 3.8 
- Conda or Virtualenv
- Jupyter Notebook (for Exploratory Data Analysis)

## Up & Running 

- Set up an environment variable named `TMDB_API_KEY`

- Exploratory Data Analysis (EDA):
```
$ pip install requirements.txt
$ jupyter lab
```
